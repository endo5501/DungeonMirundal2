# Change: ダンジョンUI

## Summary

ダンジョン画面にオートマッピングのミニマップとパーティメンバー表示枠をオーバーレイUIとして追加する。

## Scope

- オートマッピング（ミニマップ）
  - 画面右上に半透明オーバーレイとして表示
  - 探索済み領域を上から見た2Dマップとして描画（Image + TextureRect 方式）
  - エッジベース描画: WizMapの辺（WALL/OPEN/DOOR）に基づいて壁・ドア・通路を表現
  - 現在位置と向きの表示
  - 未探索領域は描画しない（黒/透明のまま）
  - DungeonViewの可視セルも探索済みとして記録
- 探索済み管理（ExploredMap）
  - 訪れたセル＋視界に入ったセルを追跡する独立クラス
  - 壁情報は持たず、描画時にWizMapから取得
- パーティ表示枠
  - 画面下部に半透明オーバーレイとして表示
  - 前列3名・後列3名のレイアウト
  - 各枠: プレースホルダー画像、名前、LV、HP(現在/最大)、MP(現在/最大)
  - 仮データでの表示（character-and-party-system 完了後に実データに差し替え）
- レイアウト方式
  - 3Dダンジョンビューはフルスクリーン維持
  - ミニマップ・パーティ枠はすべてオーバーレイ（レイアウトA）

## Non-goals

- パーティメンバーの実データ連携（character-and-party-system で対応）
- 戦闘UI
- 設定メニュー
- ミニマップのズーム/スクロール機能

## Dependencies

- dungeon-3d-rendering-and-movement: ダンジョン3D描画 + 移動（ダンジョンシーンの基本構成）

## Risks

- ミニマップのピクセルスケーリングとUI上のサイズ感調整
- 半透明オーバーレイと3D描画の視認性バランス
