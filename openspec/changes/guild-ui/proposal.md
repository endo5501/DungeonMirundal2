# Change: 冒険者ギルドUI

## Summary

冒険者ギルドの各画面（キャラクター作成、パーティ編成、キャラクター一覧、パーティ名変更）を実装する。character-and-party-system で構築したロジック層の上にUI/シーン層を構築する。

## Scope

- 冒険者ギルドメニュー画面（施設内メニュー選択）
- キャラクター作成画面
  - 名前入力
  - 種族選択（ステータス基礎値の表示）
  - ボーナスポイント配分UI
  - 就任可能な職業の表示・選択
  - 作成確認
- パーティ編成画面
  - 前列3名・後列3名の配置
  - ギルド待機中キャラクターとの入替え
- キャラクター一覧画面
  - 全キャラクター（パーティ所属・待機中）の一覧表示
  - ステータス詳細表示
  - キャラクター削除（確認ダイアログ付き）
- パーティ名変更画面

## Open questions（設計時に決めること）

- 画面遷移フロー（ギルドメニュー ↔ 各画面の遷移構造）
- town-screen-and-navigation との統合方法（地上画面の一部として動くか、独立シーンか）
- キャラクター作成画面のステップ構成（1画面 or ウィザード形式）
- ボーナスポイント配分のUI操作方法（+/- ボタン？ スライダー？）
- キャラクター一覧のソート・フィルタ要否
- 削除確認のUI仕様

## Non-goals

- ロジック層の実装（character-and-party-system で完了済みの前提）
- 商店・教会などギルド以外の施設UI（別change）
- 地上画面全体の画面遷移管理（town-screen-and-navigation）

## Dependencies

- character-and-party-system: キャラクター・パーティのロジック層
- town-screen-and-navigation との連携（並行 or 後続）
