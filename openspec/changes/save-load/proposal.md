# Change: セーブ/ロード

## Summary

ゲームの状態を保存・復元する仕組みと、設定画面（ESCメニュー）を実装する。

## Scope

- セーブデータ構造設計
  - パーティ情報
  - キャラクター情報（ステータス、装備等）
  - ダンジョン状態（生成済みマップ、探索状況、現在位置）
  - ゲーム進行状態
- 保存・読込ロジック
  - ファイルへの保存
  - ファイルからの読み込み
  - 複数セーブスロット対応
- スタート画面との統合
  - 「前回から」機能
  - 「ロード」でセーブデータ選択
- 設定画面（ESCメニュー）
  - パーティメニュー（ステータス表示、アイテム操作※枠のみ、装備管理※枠のみ）
  - ゲームを保存
  - ゲームをロード
  - 設定
  - 終了
  - 地上・ダンジョン両方から呼び出し可能

## Non-goals

- アイテム操作・装備管理の実ロジック（items-and-economy）
- 設定項目の詳細（音量、キー設定等は後続で必要に応じて）

## Dependencies

- town-screen-and-navigation: 地上画面 + 画面遷移（スタート画面、画面遷移管理）

## Risks

- セーブデータのバージョニング（将来の互換性）
- ダンジョン状態のシリアライズ（マップ全体の効率的な保存形式）
